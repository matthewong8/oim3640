## My Project Proposal

**What I'm building:** (one sentence)
- I am building a web app that allows users to enter any location in Boston and find the nearest MBTA stop and other information like accessibility.
**Why I chose this:** (personal motivation - what problem does this solve for you?)
- A lot of people use public transportation and it is a very common mode of transportation. It can be confusing to find which stop is closest or which stop is the right stop to reach your destination especially when you may be unfamiliar with the area. Therefore, this web app wil makeit much easier for citizens to navigate the MBTA system and get real guidance from the web app.
**Core features:** (3-5 bullet points of what it will do)
- take user input (address or name of place)
- Finding the nearest MBTA stop through the MBTA API
- Display the stop name
**What I don't know yet:** (honest list of things you'll need to figure out)
- Perhaps how to clearly integrate the API
- How to extract the correct stop information
- How to connect everything into a flask web app with results on display
